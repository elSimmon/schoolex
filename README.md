## About SchooLex

SchooLex is a very powerful school management system designed for secondary in Nigeria, It is tailor made for schools and encompasses features like:

- Simple, fast school fees payment system.
- Powerful Security and Access Control System.
- Multiple Admin User Levels and priviledges for (Principal, Admission Officer, Tutor, Payor, etc.)
- Result upload and result checking.
- Admission Portal.

SchooLex is a user friendly, powerful, and robust school portal required for large, and small schools.
